## 概要
[**DXライブラリ**](https://dxlib.xsrv.jp/index.html)を使用したミニゲームとなっています。

ヘッダーファイルを使用して、各キャラクターの操作部分を分割しています。

## 操作方法

1. **全面共通**

**Player**は**←↑→**で**Player**を動かします。

**Player**の**HP**が**0**になったら、**GAMEOVER**です。

2. **1面**

**Monster**にあたると**Player**の**HP**と**Defense**が減少。

**Block**の下でジャンプすると、**Block**にぶつかった後に**Gravity**(重力)が働く設定になっています。

3. **2面**

**Monster**の攻撃にあたると、**Monster** の **Attack**(攻撃力)から **Player**の **Defense**(防御力)を引いた分、**HP**が減少。

更に、**Monster**の攻撃にあたると、**Player**はヒットバックします。

また、**Monster**本体にあたると、即**GAMEOVER**です。

赤い**Block**は耐久度をもった**Block**になります。

そのため、**Monster**の攻撃が当たると**Block**のカウント変数を加算し、設定した耐久度(ヒット回数)を超えるとBlockは消滅します。

各**Block**を配列情報で設定しているので、**Block**は一つずつに耐久度が存在します。

4. **3面**

**Player**も攻撃できるようになります。

攻撃があたると、**Monster**の**HP**を下げるだけでなく、ヒットバックさせることができます。

**Monster**にあたると即**GAMEOVER**で、**Monster**を倒すと**GAMECLEAR**です。

動く**Block**には乗れることができ、**Monster**の移動から回避することができます。

しかし、**Block**はフレームアウトしますが、**Player**はフレームアウトしないので、**Block**上に居続けても、**Block**から落ちる仕組みになっています。

### 注意
拡張子が*.cppになっていますが、使用言語はCのみです。
